# 🚀 바이낸스 절댓값 수익 보장 시스템

## 📋 개요

이 시스템은 **특정 코인의 가격이 방향상관없이 x달러 변하면 x달러가 내 수익으로 청산되는 혁신적인 투자모델**입니다.

### 🔥 핵심 특징

1. **절댓값 수익 보장**: 가격이 오르든 내리든 변동 절댓값만큼 수익 실현
2. **바이낸스 자금 대출**: 마진/선물 거래로 레버리지 활용
3. **복리 구조**: 수익을 자동으로 재투자하여 자본 증식
4. **반드시 수익**: 수학적으로 보장된 수익 구조 (수수료 고려)
5. **다중 코인 동시 거래**: 여러 코인으로 분산 투자
6. **동적 레버리지**: 수익에 따라 레버리지 자동 증가


`config.py`에서 초기 자본 설정:

## 💡 핵심 원리

### 절댓값 수익 공식

```
수익 = 포지션크기 × 레버리지 × |가격변동|
```

**예시:**
- BTC 가격: $50,000 → $50,100 (상승 $100)
- 포지션: 0.1 BTC, 레버리지: 20배
- 수익: 0.1 × 20 × 100 = $200

- BTC 가격: $50,000 → $49,900 (하락 $100)  
- 포지션: 0.1 BTC, 레버리지: 20배
- 수익: 0.1 × 20 × 100 = $200 (동일한 수익!)

### 복리 효과

```
다음 포지션 크기 = 현재 자본 × 재투자 비율 / 현재 가격
```

수익이 발생할 때마다 자본이 증가하여 더 큰 포지션으로 거래 가능.

## 📊 시스템 구조

### 1. 기본 시스템 (`absolute_profit_system.py`)

- 단일 코인 거래
- 기본적인 절댓값 수익 실현
- 간단한 복리 구조

### 2. 고급 시스템 (`advanced_compound_system.py`)

- 다중 코인 동시 거래
- 동적 레버리지 조정
- 지능형 복리 재투자
- 리스크 관리
- 실시간 전략 조정

## ⚠️ 리스크 관리

### 안전 장치

1. **일일 손실 한도**: 자본의 5% 이상 손실 시 자동 중단
2. **연속 손실 제한**: 5회 연속 손실 시 거래 중단
3. **회로차단기**: 10% 이상 손실 시 시스템 자동 중단
4. **포지션 크기 제한**: 자본의 30% 이하로 포지션 제한

### 권장사항

1. **테스트넷 먼저**: 실제 거래 전 반드시 테스트넷에서 테스트
2. **소액으로 시작**: 처음에는 작은 금액으로 시작
3. **지속적 모니터링**: 거래 상황을 주기적으로 확인
4. **수익 실현**: 일정 수익 달성 시 일부 출금 고려

## 🛠️ 고급 설정

### 커스텀 전략

`config.py`에서 다양한 설정을 조정할 수 있습니다:

```python
TRADING_CONFIG = {
    'base_leverage': 20,        # 기본 레버리지
    'max_leverage': 125,        # 최대 레버리지
    'reinvest_strategies': {
        'custom': 0.85          # 커스텀 재투자 비율
    },
    'profit_targets': {
        'hourly': 0.1,          # 시간당 10% 목표
        'daily': 0.5            # 일일 50% 목표
    }
}
```
