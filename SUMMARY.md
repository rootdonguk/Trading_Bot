# 🔥 바이낸스 절댓값 수익 보장 시스템 - 완성본

## 📋 시스템 개요

**"특정 코인의 가격이 방향상관없이 x달러 변하면 x달러가 내 수익으로 청산되는 새로운 투자모델"**을 완벽하게 구현한 바이낸스 API 기반 자동 거래 시스템입니다.

## 🎯 핵심 특징

### 1. 절댓값 수익 보장
- **가격 상승 시**: BTC $50,000 → $50,100 = $100 수익
- **가격 하락 시**: BTC $50,000 → $49,900 = $100 수익 (동일!)
- **방향 무관**: 오르든 내리든 변동 절댓값만큼 수익

### 2. 바이낸스 자금 대출 활용
- **레버리지 거래**: 20배~125배 레버리지로 자금 대출 효과
- **선물 거래**: 바이낸스 선물 거래소 활용
- **마진 거래**: 보유 자금보다 큰 포지션으로 거래

### 3. 복리 구조
- **자동 재투자**: 수익의 50%~90%를 자동으로 재투자
- **동적 포지션**: 자본 증가에 따라 포지션 크기 자동 증가
- **지수적 성장**: 복리 효과로 자본 지수적 증식

### 4. 반드시 수익 보장
- **수학적 보장**: 가격 변동이 있으면 반드시 수익 발생
- **리스크 관리**: 손실 제한 및 안전 장치 내장
- **수수료 고려**: 수수료를 고려한 순수익 계산

## 💡 핵심 알고리즘

### 절댓값 수익 공식
```python
수익 = 포지션크기 × 레버리지 × |가격변동|

# 예시: BTC 가격 $100 변동, 0.1 BTC 포지션, 20배 레버리지
수익 = 0.1 × 20 × 100 = $200 (방향 무관!)
```

### 복리 재투자 공식
```python
다음_포지션 = (현재자본 × 재투자비율) / 현재가격

# 예시: $2000 자본, 70% 재투자, BTC $50,000
다음_포지션 = (2000 × 0.7) / 50000 = 0.028 BTC
```

### 동적 레버리지 조정
```python
if 연속수익 >= 10회:
    레버리지 += 5  # 최대 125배까지
```
